-> Sistema de Banco de Dados - Locadora

Projeto de estudo com foco em modelagem de banco de dados e consultas SQL para simular um sistema de locadora.

-> Funcionalidades

- Criação de banco de dados relacional
- Modelagem de tabelas com chaves primárias e estrangeiras
- Inserção de dados
- Consultas SQL (SELECT, IN, NOT IN, MAX)
- Atualização de dados (UPDATE)

-> Estrutura

O sistema inclui:
- Clientes
- Funcionários
- Pedidos
- Produtos

-> Tecnologias

- SQL

-> Como executar

1. Execute `schema.sql`
2. Execute `inserts.sql`
3. Execute `queries.sql`
